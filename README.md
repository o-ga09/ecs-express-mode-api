## ECS Express Mode API

ECS Express Modeがリリースされた記念にAPIをデプロイして使ってみるリポジトリ

### 概要

ECS Express ModeのAPIをGo言語で実装し、AWS ECS上にデプロイするサンプルコードです。

### 使用技術

|名称|バージョン|
|---|---|
|Go|1.25.3|
|Docker|24.0.5|
|AWS ECS|ECS Express Mode|
|golangci-lint|v2.4.0|
|octocov|v0.1.0|

### Get Started

- Build

```
make build
```

- Run

```
make run
```

- Test

```
make test
```

- Lint

```
make lint
```
